How to Install Eaglercraft

WASM & JavaScript Versions

Eaglercraft is a web-based version of Minecraft that you can play entirely in your browser.

There are two builds: WebAssembly (WASM) and JavaScript (JS).

Why Two Builds?

  • WASM Build: Runs faster and uses modern WebAssembly. Ideal for better performance—if your browser supports it.
  • JS Build: Uses standard JavaScript. Compatible with older setups or certain scenarios where WASM can’t run.

Enable JSPI Flags (if needed)

Some browsers require enabling JSPI (JavaScript Interface) flags for certain advanced or experimental features in Eaglercraft:

  1. Open your browser's flags or experimental features page.
  2. Search for javascript experiments or JSPI.
  3. Set it to Enabled (or Default if that includes experimental JS features).
  4. Restart your browser if prompted.

This step may differ by browser, so consult your browser’s documentation if you have trouble.

Steps to Install the WebAssembly Version

  1. Download the WASM package from the download section below.
  2. Extract the ZIP file to a folder of your choice.
  3. Double-click or open the index.html file inside that folder—this launches Eaglercraft in your default browser.
  4. If you see any warnings about WebAssembly or JSPI flags, be sure to follow the steps above to enable them.

Steps to Install the JavaScript Version

  1. Download the JS build from the download section below.
  2. Extract the ZIP file anywhere you like.
  3. Open index.html to play in your browser. This version is slightly less performant but doesn’t require advanced WASM support.
